Downloading Stock Data Using Python Code:¶

In this document, we will learn how to obtain US stock data using Python. The plan with this Jupyter Notebook is to break each step and explain each line of command. That way you will have an understanding of the whole process.

Unlike a regular Python file script, Jupyter Notebook allows us to run each line of code separately.

Loading the necessary packages¶

The most important package we need is the package that allows us to download data from Yahoo Finance. This package is called yfinance. By default, the package is not available in the Anaconda Distribution Platform, so if you are using this package for the first time on your computer, you need to run the following code in Anaconda: pip install yfinance

The rest of the packages are usually available and should load with no issues.

Basic Functions¶

To use the yfinance package, we call it by the nickname we gave it (yf). Then we can use many of its extensions. One important extension is (download). Download allows us to obtain stock data for any security available in Yahoo Finance.

However, the extension download requires many inputs or additional information. First, download what? We have to write down the name of the stock ticker (or stock tickers). In the following example, we will download Microsoft stock prices. The ticker for Microsoft is MSFT.

The required additional information is related to the type of stock prices we want for MSFT. In this example, we will download daily stock prices, so we write interval = "1d". If we want monthly prices, i.e., the price of MSFT at the beginning of each month, we replace "1d" with "1mo". Here are the following intervals you can use:

  • Every minute "1m"

  • Every 2 minutes "2m"

  • Every 5 minutes "5m"

  • Every 15 minutes "15m"

  • Every 30 minutes "30m"

  • Every 60 minutes "60m" or "1h"

  • Every 90 minutes "90m"

  • Every day "1d"

  • Every 5 days "5d"

  • Every week "1wk"

  • Every month "1mo"

  • Every 3 months (a quarter) "3mo"

The input period determines how far back we download prices. For example we will decide to download daily prices for the last month. Thus: period = "1mo", interval = "1d". For period, you are not restricted to the above list, so you can choose any integer. Further, you can choose years "y", e.x, 5y meaning the last 5 years.

Let us try now:

Visualizing Returns¶

Before we have a look at how a particular stock's return behaved in the past, we need to find a way to calculate stock returns in our datasets. Notice that our tables have the stock price dates arranged in an ascending order (i.e., from oldest to the most recent date). Thus, what we need to do is for each row, we would like to do the following formula: $$Return_{t} = \frac{P_{t}-P_{t-1}}{P_{t-1}}$$ To do so, I will use one of the many features of the Pandas package, which is pct_change. This will calculate the percentage change from the previous row within the table.
