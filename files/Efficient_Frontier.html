Finding the most efficient portfolio¶

Now with the help of our computer, we will consider ALL the different sets of weights possible for the 8 stocks to reach a portfolio that has the highest Sharpe ratio. That is, the highest return per unit of risk

To do so, we need to give a series of instructions to the computer to A) choose a random set of weights each time then b) create a portfolio with them then c) measure the risk then d) calculate the Sharpe ratio. Then finally, save it in a list so it can be compared with all other portfolios made through the same process.

main_list = [] 

for i in range(10000000):
    
    weight = np.array([np.random.random_sample() for x in stock_names])
    weight /= np.sum(weight)
    entry = [weight.dot(stock_means)]
    entry.append(weight.dot(stock_cov).dot(weight.T)**0.5)
    sharpe = (entry[-2] - r_f ) / entry[-1]
    entry.append(sharpe)
    main_list.append(np.append(weight, entry))
